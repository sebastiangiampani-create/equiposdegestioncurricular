## Uso

1. Subir `index.html`, `style.css` y `script.js` a la raiz del repositorio.
2. Crear la carpeta `audio` y subir los seis archivos `.m4a`.
3. Crear la carpeta `documentos` y subir `del-pci-al-plan.pdf`.
4. Activar GitHub Pages desde Settings > Pages.
